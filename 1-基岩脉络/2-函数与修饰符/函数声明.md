# 函数声明

```solidity

// <> 表示可选
function 函数名称(<参数1类型 <存储位置修饰符> 参数1名称, ...>) {internal|external|public|private} [pure|view|payable] [virtual|override] [<modifiers>]
[returns (<返回值类型 存储位置修饰符>)]{ <function body> }

// 注意 1：合约中定义的函数需要明确指定可见性，它们没有默认值。
// 注意 2：public|private|internal也可用于修饰状态变量。
// 注意 3：public变量会自动生成同名的getter函数，用于查询数值。
// 注意 4：未标明可见性类型的状态变量，默认为internal。

```

- 在以太坊中，以下语句被视为修改链上状态:

1. 读取状态变量。
2. 释放事件。
3. 创建其他合约。
4. 使用 selfdestruct。
5. 通过调用传送以太币。
6. 调用任何未标记view或pure的函数。
7. 使用低级调用（低级调用）。
8. 使用包含某些操作码的内部关联组件。

- view函数可以读取状态变量，但不能改写；pure函数既不能读取也不能改写状态变量。
